UFO Sightings Visualization

Plot 1: UFO Sightings in top 10 states from 1980 onwards

The above visualization is my altair plot for the number of UFO Sightings over time for the top 10 U.S. states with the highest number of reported sightings. The x-axis in my plot represents the year (from 1980 onwards) and the y-axis represents the number of reported sightings for that year. Each of the 10 colored lines on the graph represent one of the top 10 states we are plotting, which allows the audience to compare sightings by year for the top 10 states easily.

In terms of the design choices for this plot, I decided to go with a line graph, since we are plotting data over time. Technically, a bar graph could have been used as well, but it would have been far more messy and unreadable, since there would have been 10 bars for each year mark, which influenced my decision to use a line graph instead. The x-axis of this plot is encoded as an ordinal variable representing years, and the y-axis of the plot is encoded as a quantitative variable of counted sightings. As mentioned previously, Color is mapped to the categorical variable state, with distinct colors to distinguish the 10 states I was plotting, and a legend to the right side indicating which color corresponded to which state. The usage of distinct colors makes it very easy for the audience of my visualizations to quickly tell apart differences in trends of UFO sightings for different US states, and is a clear and easy way to represent different subsets of the data in this plot.

As for transformations done on the data for this plot, the first transformation was to add column names to the dataset so that each column could be easily referred to. Since the raw data did not have column names, these were added in after the csv file was read in, and were important to both plot 1 and plot 2, since both plots referred to the column names extensively.

Additionally, each date_seen timestamp was converted into a pandas datetime type, and the year was then extracted from that column so we were able to refer to sighting years instead of days or months. This helped us see broader trends in the data over a large period of time, instead of having it in a broken up way where trends are almost impossible to analyse.

Another transformation that was made for plot 1 and 2 was to drop NA values from the state and country columns in our dataset. Null state values are useless when trying to compare different sighting metrics across US states, so it did not make sense to keep these entries for our plot.

I also decided to modify the data to only capture sightings from 1980 onwards, instead of all sighting years in the dataset. The reason for this decision is because I noticed that prior to 1980, there was not alot of data regarding sightings for any of the top 10 states, and the limited number of sightings seemed to be pretty similar for all states being plotted. Therefore, it did not make sense to keep these years in the plot, as they were unnecessarily making the plot bigger while not adding any meaningful information, and taking away from the important trends and differences in counts we can see post-1980.

Lastly, I decided to make a transformation in the data to only limit the top 10 states by sighting count from being plotted in our graph. This is because, similar to the reason for the 1980-onwards transformation, having all 50 states on the graph would make it cluttered without much benefit, since most states have very low number of sightings.

Plot 2: UFO Sightings by Country from 1980 onwards

For this visualization, I have decided to show UFO sightings over time across different countries, using the x-axis to represent year and using the y-axis to represent the count of sightings. The lines on the graph represent a country, and users are able to swtich which country they want to view data for by using the dropdown menu at the bottom of the graph. Since the US has the most data about UFO sightings, I have kept the US as the default country.

Similar to the first plot, the encodings of the second plot include the year mapped to the x-axis, and the count of UFO sightings mapped to the y-axis. In this plot, I chose not to use the color encoding, as each country is distinctly selected using the dropdown, so there is no instance where more than one country is displayed at a time on the graph. A line chart is appropriate for this graph as well because we want to observe trends over time, which are best depicted through a line graph.

As for transformations, transformations to the data for this plot were already done for plot 1, which prevented me from having to do them again. These include adding column names to the dataset, converting dates to datetime format and adding a year column, dropping NA values for the country column, and capturing data from only 1980 onwards due to the reasons listed in the description for plot 1.

For the additional interactivity component for this plot, I decided to create a dropdown using which users can select which country they want to view UFO sighting data for. The dropdown includes all countries in the dataset (non-NULL), and selecting any country shows a trend-line of UFO sighting data for that corresponding country. This helps make my visualization more interesting because it gives users the ability to focus in on individual countries of interest instead of having to see them all together. Without this interactive dropdown component, our alternatives would be to either create a plot with all countries data (similar to plot 1 but with countries instead of states), or to create a different plot for each country. Both options are inconvenient, with the first one not allowing users to focus on a singular country's data, and the second one requiring lots of unnecessary space and code. The use of this dropdown is even more important when dealing with data which has more fields, such as city data. There are likely hundreds of different cities in the provided dataset, and it would not be practical to give a graph with all of them or create individual graphs. In those cases as well as my country example, interactive components such as dropdowns are incredibly effective.
